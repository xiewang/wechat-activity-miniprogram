# 图片资源目录

本目录用于存放小程序所需的图标和图片资源。

## 需要的图标文件

请将以下图标文件放入此目录：

### TabBar 图标（建议尺寸：81x81px）
- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标
- `search.png` - 搜索未选中图标
- `search-active.png` - 搜索选中图标
- `info.png` - 免责声明未选中图标
- `info-active.png` - 免责声明选中图标

### 图标建议
- 使用简洁的线性图标风格
- 颜色：未选中 #999999，选中 #4CAF50
- 格式：PNG
- 尺寸：81x81px

可以使用阿里巴巴矢量图标库（iconfont.cn）下载合适的图标。
