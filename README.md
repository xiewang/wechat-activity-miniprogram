# 微信活动小程序

## 项目简介

本项目旨在构建一个微信小程序，自动收集并展示上海地区面向中小学生、幼儿等的课内外活动、兴趣活动等信息。主要功能包括：

1. **每日自动抓取** 活动信息（爬虫或第三方 API）
2. **活动分类** 与分组展示
3. **搜索过滤**（按类型、地区、年纪、人群等）
4. **活动详情**，包括截至日期、主办方联系方式
5. **法律风险提醒** 与免责协议
6. **后台审核** 防止非法或误发布的内容
